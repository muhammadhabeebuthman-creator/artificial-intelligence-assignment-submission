# Missionaries and Cannibals - AI Group Project

Classic AI search problem solved with multiple algorithms + visualizations.

## Features
- 5 search implementations (BFS, DFS, Backtracking, A*, State Space Graph)
- Interactive **Pygame** animation (play yourself!)
- **Matplotlib** animation of recursive DFS solution steps + exploration tree
- Time & Space complexity comparison (DFS vs BFS)
- Presentation slides & graphical state space representation

## How to Run

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Run search algorithms

```bash
python src/search_algorithms/bfs.py
python src/search_algorithms/dfs.py
```

### 3. Run interactive Pygame animation

```bash
cd animation/pygame
python main.py
```

### 4. See recursive solution animation (Matplotlib)

```bash
python animation/matplotlib/recursive_visualization.py
```

### 5. See complexity comparison

```bash
python src/complexity_analysis.py
```

### Project Structure

```bash
src/                  → AI search algorithms
animation/            → Visualizations (Pygame + Matplotlib)
docs/                 → Documentation & slides
assets/               → Screenshots
```
